📌 Project Overview

This project focuses on analyzing Google search trends to understand the popularity of different technologies over time. The analysis highlights search interest patterns, regional trends, and keyword comparisons for Cloud Computing, Data Science, and Machine Learning.

🎯 Objectives

Analyze search interest over time

Identify top regions searching for cloud computing

Compare popularity of different tech keywords

Visualize trends and patterns in search behavior

🛠️ Tools & Technologies Used

Python

Pandas

NumPy

Matplotlib

Seaborn

Google Trends (Pytrends API)

📂 Dataset Description

The dataset was collected using Google Trends and includes:

Search interest over time

Region-wise search data

Keyword comparison metrics

🔍 Key Analysis Performed

Data collection using Google Trends API

Data cleaning and preprocessing

Time series analysis of search trends

Region-wise analysis to identify top searching locations

Comparative analysis of multiple keywords

📊 Key Insights

📍 Top Region: St. Helena shows the highest search interest for Cloud Computing

📈 Search Interest Over Time: Peak search interest observed in 2026

🔄 Keyword Comparison:

Cloud Computing shows strong and consistent growth

Data Science maintains steady popularity

Machine Learning shows rising interest over time
